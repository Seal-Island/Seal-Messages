# Seal Forge Messages
Seal Forge Messages é um mod para Minecraft 1.12.2 que permite a modificação de algumas mensagens hard-coded do Forge.

Sendo elas:
 - `Server is still starting! Please wait before reconnecting.`
 - `This server has mods that require FML/Forge to be installed on the client. Contact your server admin for more details.`
